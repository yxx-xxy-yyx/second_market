# 介绍

EchoOfMemories UI 是一个为校园二手交易平台量身定制的 Vue 3 组件库，基于 Element Plus 构建，提供完整的设计系统和丰富的UI组件。

## ✨ 特性

- 🎨 **统一的设计系统** - 完整的设计令牌，确保视觉一致性
- 📱 **响应式设计** - 自动适配桌面、平板、移动端
- ⚡ **性能优化** - 按需导入，Tree-shaking 支持
- ♿ **无障碍支持** - 遵循 WCAG 标准
- 🌙 **暗黑模式** - 一键切换主题
- 📖 **完整文档** - 详尽的API文档和使用示例

## 🏗️ 技术栈

- Vue 3 (Composition API)
- Element Plus
- Tailwind CSS
- Vite
- TypeScript (可选)

## 💡 设计理念

我们的设计理念是 **"简洁、高效、一致"**：

1. **简洁** - 去除不必要的复杂性，提供直观易用的组件
2. **高效** - 优化性能，减少打包体积，加快加载速度
3. **一致** - 统一的设计语言，确保整个应用的视觉一致性
