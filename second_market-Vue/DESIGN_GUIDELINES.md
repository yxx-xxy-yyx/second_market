# 校园二手交易平台 - 设计规范文档

## 一、设计理念

本项目采用**简约现代**的设计风格，以用户体验为中心，追求视觉美感与功能实用的完美平衡。

### 1.1 设计目标

- **简洁高效**：界面清爽，信息层次分明，操作路径短
- **现代感强**：运用毛玻璃、渐变、阴影等现代设计元素
- **响应式设计**：完美适配桌面、平板、移动端等多种设备
- **国际化支持**：中、英、韩三国语言无缝切换

### 1.2 核心设计原则

1. **一致性原则**：统一的色彩、字体、间距、圆角等设计变量
2. **层次清晰**：通过明暗、大小、色彩对比建立信息层次
3. **可触达性**：考虑各种用户群体的使用习惯
4. **性能优先**：轻量化设计，保证流畅的用户体验

## 三、组件规范

### 3.1 按钮

#### 主要按钮

```html
<el-button type="primary">主要操作</el-button>
```

- 背景：`--primary-500`
- 文字：白色
- 圆角：`--radius-md`
- 悬停：背景变深，阴影增强
- 激活：轻微下沉效果

#### 次要按钮

```html
<el-button>次要操作</el-button>
```

- 背景：透明
- 边框：`--border-primary`
- 文字：`--text-primary`
- 悬停：背景填充

#### 危险按钮

```html
<el-button type="danger">删除</el-button>
```

- 背景：`--error-500`
- 用于不可逆操作

### 3.2 卡片

```html
<div class="card-common">
  <!-- 卡片内容 -->
</div>
```

- 背景：`--bg-card`
- 圆角：`--radius-lg`
- 阴影：`--shadow-md`
- 悬停：阴影增强，轻微上浮

### 3.3 输入框

```html
<el-input placeholder="请输入..." />
```

- 圆角：`--radius-md`
- 边框：`--border-primary`
- 聚焦：边框变为主色

### 3.4 对话框

```html
<el-dialog title="提示" v-model="visible">
  <!-- 内容 -->
</el-dialog>
```

- 圆角：`--radius-xl`
- 背景遮罩：`rgba(0,0,0,0.5)`

## 六、无障碍设计

### 6.1 焦点状态

所有交互元素都有清晰的焦点状态：

```css
:focus-visible {
  outline: 2px solid var(--primary-500);
  outline-offset: 2px;
}
```

### 6.2 减少动画

尊重用户的减少动画偏好：

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

### 6.3 颜色对比度

所有文字都满足 WCAG AA 标准（至少 4.5:1）。

## 七、最佳实践

### 7.1 样式优先级

1. 使用设计令牌（CSS变量）
2. 使用Tailwind工具类
3. 使用自定义CSS（必要时）

### 7.2 组件开发

1. 使用组合式API（Composition API）
2. 组件命名采用PascalCase
3. Props定义完整类型
4. 提供合理的默认值

### 7.3 性能优化

1. 组件懒加载
2. 图片优化
3. 代码分割
4. 资源压缩
